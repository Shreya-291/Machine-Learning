# SVM Hyperparameter Tuning: Exploring C and Gamma

This repository contains a machine learning tutorial demonstrating how hyperparameters influence the performance of Support Vector Machines (SVM).

## Overview

The tutorial investigates the effect of two important SVM hyperparameters:

• **C (Regularization parameter)** – controls the trade-off between margin size and classification errors.  
• **Gamma (Kernel coefficient)** – controls the complexity of the decision boundary.

Using the **Iris dataset**, the tutorial visualizes how different values of these parameters affect decision boundaries and model performance.

## Files in this Repository

- **svm_hyperparameter_tuning.ipynb** – Jupyter Notebook containing the implementation and visualizations  
- **tutorial_report.pdf** – Tutorial report explaining the experiment and results  


## Requirements
